API Documentation

REST API reference for the Synthetix backend

POST
/api/v1/analyze

Analyze a defect report for duplicates, assign cluster, and enhance.

Request Body

{
  "defect_id": "BUG-2001",
  "title": "Login fails with expired token",
  "description": "When user tries to log in...",
  "steps": "1. Open login page\n2. Enter credentials",
  "expected": "Should redirect to dashboard",
  "actual": "500 Internal Server Error",
  "environment": "Chrome 120, Windows 11",
  "logs": "NullPointerException at..."
}

Response

{
  "decision": "duplicate",
  "top_matches": [
    {
      "defect_id": "BUG-1001",
      "title": "Login page crashes with expired JWT",
      "similarity_score": 0.9234,
      "cluster_id": 0
    }
  ],
  "cluster_id": 0,
  "improved_report": {
    "improved_title": "Login fails with expired JWT token - 500 error",
    "summary": "Authentication module fails to handle...",
    "missing_fields": [],
    "completeness_score": 90.0
  },
  "confidence": 0.9234
}
POST
/api/v1/ingest

Upload and ingest a CSV/JSON defect dataset.

Request

curl -X POST http://localhost:8000/api/v1/ingest \
  -F "file=@data/sample_defects.json"

Response

{
  "total_ingested": 20,
  "total_skipped": 0,
  "clusters_formed": 5,
  "message": "Successfully ingested 20 defects..."
}
GET
/api/v1/clusters

Get overview of all defect clusters.

Response

{
  "total_defects": 20,
  "total_clusters": 5,
  "noise_count": 3,
  "clusters": [
    {
      "cluster_id": 0,
      "size": 4,
      "representative_title": "JWT auth failure",
      "defect_ids": ["BUG-1001", "BUG-1002", ...]
    }
  ]
}
